There will/may be two versions of Overleaf (Latex) file, use your preferable one.
Download the Zip file to use as your Thesis Proposal Formal and import to Overleaf.

To be compatiable with iThesis for your Proposal or Thesis submission, you might need to change the following command lines.
(1) Compile your latex code
(2) Go to Logs, scroll down to the bottom, and click other logs and files to download output.bbl and output.aux
(3) Import (Drag and Drop) to your latex folder and complie again (you may skip the compile here)
(4) change you bib file name in the latex code as \bibliography{main}
(5) Compile and check your PDF file again
